========================================================================
    КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ. Обзор проекта parser
========================================================================

Программа принимает на вход 4 параметра:
- Имя входного файла с данными для фильтрации;
- Имя выходного файла куда будут записываться отфильтрованные данные;
- Имя конфигурационного файла с ключевыми фразами (фильтрами);
- bool переменная определяющая режим работы (запись строк которые содержат(1)/не содержат(0)  заданных сочетаний символов).
Фильтры из  конфигурационного файла  разделяем на 2 группы состоящие из 1 символа и остальные. Заносим каждую группу в отдельный  непронумерованный набор.
Проверяем каждый символ входного файла на вхождение в непронумерованный набор первой группы фильтров, что состоят из 1 символа. При совпадении записываем данную строку в выходной непронумерованный набор.
Используя вторую группу фильтров(с строками длинной более 1 символа) проверяем строки на вхождение ключевой фразы используя алгоритм Бойера-Мура. При совпадении записываем строку в выходной непронумерованный набор.
В случае выбора пользователя режима работы записи строк которые содержат заданные сочетания символов - записываем в выходной файл строки из выходного непронумерованного набора.
В случае выбора пользователя режима работы записи строк которые не содержат заданные сочетания символов - записываем в выходной файл строки из входного файла которые не содержит выходной непронумерованный набор.

Ограничение: 256 символов в строке.

В этом файле представлена сводка содержимого всех файлов, входящих в состав приложения parser.

parser.vcxproj
    Это основной файл проекта VC++, создаваемый с помощью мастера приложений. Он содержит данные о версии языка Visual C++, использованной для создания файла, а также сведения о платформах, конфигурациях и функциях проекта, выбранных с помощью мастера приложений.

parser.vcxproj.filters
    Это файл фильтров для проектов VC++, созданный с помощью мастера приложений. Он содержит сведения о сопоставлениях между файлами в вашем проекте и фильтрами. Эти сопоставления используются в среде IDE для группировки файлов с одинаковыми расширениями в одном узле (например CPP-файлы сопоставляются с фильтром "Исходные файлы").

main.cpp
    Это основной исходный файл приложения.

Hesh_find,cpp
	Это файл осуществляет определие находится ли клчевое слово в строке посредством  хеш-таблицы

BM_find.cpp 
	Это файл осуществляет определие находится ли клчевое слово в строке посредством алгоритма Бойера-Мура

parsing.cpp 
	Это файл осуществляет фильрацию строк файла.

/////////////////////////////////////////////////////////////////////////////

StdAfx.h, StdAfx.cpp
